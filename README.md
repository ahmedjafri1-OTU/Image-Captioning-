# 🖼️ Image Captioning App
An AI-powered application that generates natural language descriptions for images using the BLIP (Bootstrapped Language Image Pretraining) model from Hugging Face Transformers. The app uses PyTorch for inference and Gradio to provide a lightweight interactive interface, allowing users to upload images and instantly receive generated captions.
* ⚙️ Tech Stack
* 🐍 Python
* 🔥 PyTorch
* 🤗 Hugging Face Transformers
* 🎛️ Gradio
* 🖼️ PIL (Python Imaging Library)
  
### 🚀 Key Features
Transformer-based image caption generation
Real-time inference using a pretrained BLIP model
Simple web interface for uploading and captioning images
Demonstrates integration of computer vision + NLP models
