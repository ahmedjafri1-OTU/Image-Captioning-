# 🖼️ Image Captioning & Image Indexing Tool
A Python application that analyzes images and automatically generates descriptive captions using a vision-language model. The generated captions are stored in a text-based index file, enabling organized documentation and easier search of image datasets.
The application uses the BLIP image captioning model from Hugging Face Transformers and provides an interactive interface using Gradio.

### 🚀 Objectives
Implement an image captioning system using the BLIP model from Hugging Face
- Build a user-friendly interface with Gradio
- Automatically generate and store image descriptions in an index file
- Demonstrate how image understanding can support real-world data organization workflows

### 🧠 Technology
The project uses the BLIP (Bootstrapping Language-Image Pretraining) model, a vision-language model that combines computer vision and natural language processing to generate natural language descriptions from images.
